# 贡献指南

感谢您对Pexels 4K视频下载器项目的关注！我们欢迎任何形式的贡献。

## 如何贡献

### 报告Bug

如果您发现了Bug，请在GitHub上创建一个Issue，包含以下信息：
- 问题的清晰描述
- 重现步骤
- 您的操作系统和Python版本
- 屏幕截图（如果可能）

### 提交功能请求

如果您有新的功能想法，请创建一个Issue，描述：
- 功能的详细说明
- 为什么这个功能有用
- 可能的实现方式

### 提交代码

1. Fork项目
2. 创建您的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个Pull Request

## 开发环境设置

1. 克隆仓库
2. 确保您有Python 3.8或更高版本
3. 安装依赖: `pip install -r requirements.txt`

## 代码规范

- 遵循PEP 8代码风格
- 添加适当的注释
- 编写清晰的提交信息

## 测试

在提交代码之前，请确保：
- 所有现有功能仍然正常工作
- 新功能有适当的测试

## 问题和讨论

如果您有任何问题或想讨论功能，请在GitHub上创建Issue。

再次感谢您的贡献！
